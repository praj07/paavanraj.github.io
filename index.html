About Me

I'm a third year Mechatronics Engineering student at Western University with an interest in remote controlled robots and control system design.
In my spare time I like to design and build both remote controlled and autonomous robots. I also enjoy going to the gym, playing board games and football with friends and learning new card tricks.

Projects

  • Hex V4*
    HEX V4 is a group project that I'm currently working on in Western's Aero Design team. The purpose of HEX is data acquisition for the RC model plane made by the Aero Team. HEX is uses an inertial measurement unit and an on board computer (Raspberry pi) to store data obtained in flight and is controlled using arduino uno as the flight controller. Currently HEX is in the process of being fabricated.

  • Cat-R*
    Cat-R is a personal project that I've been working on over the summer. It is a quadrapod walking robot that uses an ATMEGA328 microcontroller and an HC05 bluetooth module so that the robot can be controlled remotely. Currently Cat-R is being remodelled to address some issues with the legs and to improve efficiency of walking.

  • Stark
    Stark was a collaborative final project for a mechatronics systems design course. The robot was designed to locate a small magnetic cube and place it under an acrylic pyramid. The creation process for Stark included creating solidworks models for fabrication, microcontroller programming to process information obtained from the sensors and to control actuators, and iterative engineering design to ensure it achieves its goal.

  • Jarvis
    Jarvis is a line following robot that uses an arduino microcontroller and was designed to follow various courses set up by the user. Additionally Jarvis has a claw end effector that is used to retrieve a light beacon and place it in the loading zone.
